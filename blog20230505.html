计算短信费

题干：假设每一条少于70字短信是0.1元，每70字是0.1元，不到按70字计算，请问总共需要多少钱？

要求：第一行输入一个整数a，表示短信数量，之后输入[短信数量]次短信字数，最后得出总价格。

请使用c++实现。




这需要使用到向上取整函数，而向上取整在c++中的用法是ceil()。其次，总字数是整数，价钱是浮点数，总价钱也是浮点数。最终代码为：


#include <bits/stdc++.h>
using namespace std;

int main() {
int char_num;
double money_sum;
int i;
int a;
double p;
money_sum = 0;
scanf("%d", &a);
for (i = 1; i <= a; i++) {
scanf("%d", &char_num);
p = ceil(char_num / 70.0) * 0.1;
money_sum += p;
}
printf("%lf", money_sum);
return 0;
}
